#  Telecom Customer Churn Prediction Dashboard

##  Overview
This project predicts customer churn using machine learning and provides insights through an interactive dashboard.

##  Features
- Data preprocessing and feature engineering  
- Exploratory Data Analysis (EDA)  
- Random Forest model (~80% accuracy)  
- Confusion matrix & performance evaluation  
- Feature importance analysis  
- Streamlit dashboard for visualization  

## 🛠 Tech Stack
- Python (Pandas, NumPy)  
- Scikit-learn  
- Matplotlib  
- Streamlit  

## Results
- Accuracy: ~80%  
- Identified key churn factors like tenure and monthly charges

pip install -r requirements.txt
